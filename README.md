# Foobar

Theming in React Native

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install and run the application.

```bash
git clone https://github.com/ganeshbhusiv/themingrn.git
```

## Usage

```javascript
npm i

npm start

react-native run-android
```
