# Foobar

Theming in React Native

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
git clone https://github.com/ganeshbhusiv/themingrn.git
```

## Usage

```javascript
npm i

npm start

react-native run-android
```
